# CoolPC 請假行事曆

一個方便查詢 CoolPC 員工請假時間的網頁應用程式。

## 功能特色

- 📅 月份瀏覽 - 可查看不同月份的請假情況
- 🏢 部門篩選 - 顯示各部門人員請假資訊
- 🎨 請假類型顏色區分
  - 🔵 排休
  - 🟢 特休
  - 🟣 喪假
  - 🔴 病假
- 📊 部門統計 - 顯示各部門請假人數

## 使用方式

1. 打開 `coolpc-calendar.html` 文件
2. 使用上個月/下個月按鈕瀏覽不同月份
3. 點擊「更新資料」重新整理

## 技術棧

- CSS3 (Flexbox + Grid)
- JavaScript (Vanilla)
